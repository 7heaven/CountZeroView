CountZeroView
=============
i use the ([TimelyTextView][2]) with only minor modifications

![Animated gif][1]


# A simple example

you just call the 

``` java
yourview.start();
```
config start from

``` java
yourview.setFromNumber(from);
```
you can countdown add end listener

``` java
yourview.setOnFinish(FinishListener);
```
